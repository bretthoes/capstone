# Message in a Bottle
A Vue.js / Express.js web application for quiz-based matchmaking and instant messaging.

Live here: https://message-in-a-bottle-client.onrender.com/#/

__Setup__

Start Client
```
cd client
npm run dev
```
Start Server
```
cd server
npm run dev
```

__Optional__
```
Create Fictional Data:
cd server
npm run seed
```
